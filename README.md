# Python Code For Practice 🐍

A collection of Python code for practice purposes — covering core concepts to interview prep.

## 📌 Topics Covered

- **Basics**: Variables & Data Types, if/elif/else, For & While loops, break/continue/pass
- **Data Structures**: Lists, Tuples, Sets, Dictionaries, Slicing & Indexing
- **Comprehensions**: List, Dict, and Dictionary Comprehensions
- **Functions**: Function Definitions, Lambda, *args & **kwargs, Decorators, Generators & yield
- **Strings**: String Methods & f-strings
- **Error Handling**: try/except/finally, Custom Exceptions, Exception Types
- **Other Concepts**: Type Hints & Annotations, len/range/enumerate/zip
- **Interview Prep**: Python for Interviews

## 🗂️ Structure

Each `.ipynb` (and a few `.py`) file focuses on a specific concept with hands-on code examples.

## 🎯 Purpose

A personal practice space for strengthening Python fundamentals — built one concept at a time.
